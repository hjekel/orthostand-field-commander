# 🇪🇸 Orthostand Field Commander

Sales trip companion app for Orthostand M30 expansion in Spain (March-April 2026).

## Features

- 📍 **Planner** — 28 contacts across 5 Spanish cities
- 📓 **Field Journal** — Visit notes with voice recording
- 🎯 **Intel** — Market insights per city
- 📊 **Dashboard** — Progress tracking & commission clock
- 📅 **Calendar** — Trip dates visualization

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- LocalStorage for data persistence

## Development

```bash
npm install
npm run dev
```

## Deployment

Hosted on Vercel. Auto-deploys on push to main.
